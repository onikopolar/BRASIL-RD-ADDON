Configuração do Real-Debrid

Como obter: real-debrid.com/apitoken
Esta é a URL do addon para instalar no Stremio

Como Instalar:

  1. Obtenha sua API key no site do Real-Debrid
  2. Cole a chave no campo acima
  3. Clique em "Instalar Addon no Stremio"
  4. Copie o link do addon que aparecerá abaixo
  5. Abra o Stremio e vá em Addons
  6. Clique em "Instalar pelo link"
  7. Cole o link e complete a instalação
  8. Configure a chave API diretamente no Stremio

Funcionalidades:

  • Conteúdo brasileiro (dublado/legendado) por: BLUDV, COMANDO, STARCK
  • Qualidade 4K, 1080p, 720p automática
  • Velocidade de busca: 2-3 segundos
  • Suporte a filmes e séries
  • Integração completa com Real-Debrid
  • Configuração individual por usuário
  • Conteúdo faltante pode ser feito request - solicite pelo Instagram: gu.dwn

Instruções por Plataforma

No Computador

  1. Abra o Stremio
  2. Clique em "Addons" no menu lateral
  3. Clique em "Instalar pelo link" no canto inferior esquerdo
  4. Cole o link do addon
  5. Siga as instruções de configuração do Stremio

No Celular

  1. Abra o Stremio
  2. Toque em "Addons"
  3. Toque no ícone "+" no canto superior direito
  4. Cole o link do addon
  5. Siga as instruções de configuração do Stremio
